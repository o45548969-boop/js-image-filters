# js-image-filters
Мій перший проєкт на JavaScript: фільтр для перетворення зображення у чорно-білі.
### 🚀 Що вміє цей проєкт:
- завантажувати зображення;
- перетворювати кольорове фото на чорно-біле (Greyscale);
- демонструвати роботу функцій JavaScript у реальному часі;
### ⚒️ Технології:
- **HTML5** - структура сторінки;
- **CSS** - стилізація інтерфейсу;
-  **JavaScript** - логіка обробки пікселів;
  ---
  *Проєкт створено в рамках самостійного вивчення програмування.*
